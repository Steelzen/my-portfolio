# my-portfolio

### Description

A personal portfolio webpage built with React.js

### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap

### Run in local

Get the code

<pre>git clone https://github.com/Steelzen/my-portfolio.git</pre>
 
Install required dependencies

<pre>yarn install</pre>

Start the server

<pre>yarn start</pre>
