# my-portfolio

### Description

A personal portfolio webpage to demonstrate personal projects.

### Tech Stack
 - React.js
 - Three.js
 - Axios
 - React Bootstrap

### Run in local

Clone the code

<pre>git clone https://github.com/Steelzen/my-portfolio.git</pre>
 
Install required dependencies

<pre>yarn install</pre>

Run the app

<pre>yarn start</pre>
